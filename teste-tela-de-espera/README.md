# TESTE FRONTEND

Este é o resultado do meu projeto de uma tela de espera de um jogo online multiplayer.

Primeiro, acho justo confessar o uso da inteligência artificial do site v0 para me auxiliar no desenvolvimento, porém o projeto não foi copiado da IA, mas sim ela foi utilizada como auxílio e para aprendizado meu, já que meu conhecimento em HTML, CSS e JavaScript no início do desenvolvimento deste projeto era mínimo demais para que eu cumprisse os requisitos, e eu não teria tempo o bastante para aprender tudo o necessário para desenvolver de maneira totalmente autoral, então utilizei a IA para poupar tempo e agilizar meu aprendizado, mas estou em busca de aprofundar meu conhecimento para conseguir desenvolver futuros projetos de maneira totalmente autoral e independente de IAs.
